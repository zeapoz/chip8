# CHIP-8 Emulator in Rust
A functional [CHIP-8](https://en.wikipedia.org/wiki/CHIP-8) emulator written in Rust with SDL.

# Instructions
To run a rom, supply the path as an argument to the executable.
```bash
$ chip8 roms/game.ch8
```
